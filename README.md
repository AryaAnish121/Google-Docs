# Google Docs

This is a google docs clone width MERN stack. Hope you will like it

## Usage/Examples

```
git clone https://github.com/AryaAnish121/Google-Docs.git

cd server
```

change the keys in .env

```
nodemon app.js
```

```
cd client/

npm start
```
